There are 3 types of lists:

defenition lists
      which show terms and defenitions
unordered lists
     which show bullets
ordered list
     which show numbers or letters

Above is an example of a defenition list.

Unorderd list can have a type of attribute
which can be applied to the UL or LI tag.
The three values allowed are:

  • type = 'square'
  • type = 'disc'
  • type = 'circle'
Ordered lists can have a type attribute too. Used along with a start attribute they can show:
  1. type = 'lower case letters'
  1. type = 'upper case letters'
  1. type = 'lower case roman numerals'
  1. type = 'upper case roman numerals'
  1. type = 'numbers'
